invert-permutation
==================
Computes the inverse of a permutation

Usage
=====
First, install using npm:

    npm install invert-permutation
    
Then, you can use it as follows:

    console.log( require("invert-permutation")([1,3,0,2]) )
    
    //Prints:
    //          [ 2, 1, 3, 0 ]
